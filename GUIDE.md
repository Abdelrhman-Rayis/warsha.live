# 🎓 شرح الموقع بالتفصيل

## البدء السريع ⚡

### 1️⃣ فتح الموقع
- اضغط على ملف `index.html` مرتين
- أو انسخ المسار في متصفح الويب

### 2️⃣ تبديل اللغة
أنقر على زر **EN/AR** في أعلى اليمين

---

## ⚙️ الخطوات الأساسية

### أولاً: إنشاء حساب

```
1. انقر على "إنشاء حساب" من الصفحة الرئيسية
2. ملء النموذج:
   - الاسم الكامل
   - البريد الإلكتروني
   - كلمة المرور
   - اختر: طالب أو معلم
3. انقر "إنشاء الحساب"
```

### ثانياً: تسجيل الدخول

```
1. انقر على "تسجيل الدخول"
2. أدخل البريد والكلمة المرور
3. انقر "دخول"
```

### ثالثاً: استعراض الكورسات

```
1. اضغط على "الكورسات" من القائمة
2. شاهد جميع الكورسات المتاحة
3. انقر على أي كورس لرؤية التفاصيل
4. انقر "التسجيل الآن"
```

### رابعاً: الدفع والتسجيل

```
1. أدخل بيانات البطاقة:
   - رقم البطاقة: 1234 5678 9012 3456
   - اسم صاحب البطاقة
   - تاريخ الانتهاء: MM/YY
   - CVV: أي ثلاث أرقام
2. انقر "تأكيد الدفع"
```

### خامساً: التفاعل مع الكورس

**📊 الملف الشخصي:**
- شاهد كورساتك
- متابع تقدمك

**💬 غرفة النقاش:**
- تواصل مع المعلم والطلاب
- اسأل الأسئلة
- شارك الأفكار

## 📞 مشاكل شائعة وحلولها

### المشكلة: الموقع لا يحمل
**الحل:**
- تأكد من فتح ملف index.html
- مسّح cache المتصفح
- حاول متصفح مختلف

### المشكلة: فقدان البيانات
**الحل:**
- البيانات مخزنة محليا فقط
- مسح السجل يمسح البيانات
- حفظ البيانات مسبقا

### المشكلة: الدفع لا يعمل
**الحل:**
- الدفع تجريبي فقط
- أدخل أي بيانات بطاقة
- تأكد من تعبئة جميع الحقول
